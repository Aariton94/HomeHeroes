## About

HomeHeroes is a web-based platform designed to motivate children to participate in household tasks through a fun and interactive reward system. Parents can assign tasks, set deadlines, and award stars. Children can complete tasks, earn stars, and exchange them for rewards.

---

## Features

### For Parents
- Create and manage child accounts
- Assign tasks with descriptions and deadlines
- Award or remove stars (Simple Stars & Super Stars)
- Create custom rewards for children to redeem
- Write notes for each child
- View task progress and deadlines

### For Children
- View assigned tasks and deadlines
- Mark tasks as completed
- Earn Simple Stars for each completed task
- Convert 10 Simple Stars → 1 Super Star
- Redeem Super Stars for rewards set by parents
- Write notes for parents

## Database Schema

<img width="670" height="372" alt="dbhh" src="https://github.com/user-attachments/assets/b57e9cbd-e5e7-40c1-b669-11d7fb728e8d" />


The application uses 5 core tables:

- `users` - Parent and child accounts
- `tasks` - Tasks assigned to children
- `stars` - Star tracking (yellow & blue)
- `rewards` - Rewards created by parents
- `notes` - Notes saved by both roles
